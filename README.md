# Repository Root

## Purpose

This repository contains the supporting documentation,  qualitative evidence, and quantitative analysis files for a dissertation on the Integrated Correction Model (ICM) for improving EFL secondary-school students' essay-writing performance.

The repository is organized into two main areas: qualitative analysis and quantitative analysis. The materials document the study context, instruments, observational evidence, coding workflow, datasets, R scripts, statistical outputs, and figures
