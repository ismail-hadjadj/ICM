# Repository Root

## Purpose

This repository contains the supporting documentation, appendices, qualitative evidence, and quantitative analysis files for a dissertation on the Integrated Correction Model (ICM) for improving EFL secondary-school students' essay-writing performance.

The repository is organized into three main areas: appendices, qualitative analysis, and quantitative analysis. The materials document the study context, instruments, observational evidence, coding workflow, datasets, R scripts, statistical outputs, figures, and supporting appendix files.

## Folder contents

### Direct files

No direct files are stored in this folder.


### Direct subfolders

| Subfolder | Purpose |
|---|---|
| `Appendices` | This folder contains the dissertation appendices extracted into separate appendix-level folders. |
| `Qualitative_Analysis` | This folder contains the qualitative component of the study. |
| `Quantitative_Analysis` | This folder contains the quantitative component of the study. |


## Recommended use

Use this repository as a structured archive of dissertation evidence and analysis materials. Keep file names stable so dissertation references remain traceable.
