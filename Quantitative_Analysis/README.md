# Quantitative Analysis

## Purpose

This folder contains the quantitative component of the study. It includes pilot-study files, main-study datasets, R scripts, descriptive statistics outputs, preliminary assumption checks, and inferential results for Hypothesis 1 and Hypothesis 2.

### Direct subfolders

| Subfolder | Purpose |
|---|---|

| `Pilot Study` | This folder contains the pilot-study dataset, R script, and descriptive summary used to examine the feasibility of the instruments and procedures before the main study.. |
