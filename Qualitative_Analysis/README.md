# Qualitative Analysis

## Purpose

This folder contains the qualitative component of the study. It includes raw observation documents, transcriptions, metadata, codebooks, QualCoder project files, coding outputs, analytical memos, cross-case matrices, and theme-integration tables.

## Folder contents

### Direct files

| File | Type | Size |
|---|---:|---:|
| `README.md` | md | 106 B |


### Direct subfolders

| Subfolder | Purpose |
|---|---|
| `01_Raw_Data` | This folder contains the original classroom observation documents for the selected control-group and experimental-group sessions. |
| `02_Transcriptions` | This folder contains text transcriptions of the structured classroom observation documents. |
| `03_Metadata` | This folder contains document identifiers and metadata describing the qualitative observation files. |
| `04_Codebook` | This folder contains the initial qualitative codebook used during the first stage of thematic coding. |
| `05_QualCoder_Project` | This folder contains the QualCoder project files and project backups used to conduct the qualitative analysis. |
| `06_First_Cycle_Coding` | This folder contains the first-cycle qualitative coding outputs. |
| `07_Refined_Coding` | This folder contains refined qualitative coding materials. |
| `08_Matrices` | This folder contains qualitative matrices and comparison tables. |
| `09_Memos` | This folder contains file-level, familiarization, and cross-case memos. |
| `10_Themes_and_Integration` | This folder contains files linking qualitative themes to quantitative findings. |


## Recommended use

Use this folder to trace the qualitative workflow from raw observation evidence to transcriptions, coding, memos, matrices, themes, and mixed-methods integration.
