Code reviewed: High_Engagement

Code reviewed: Low_or_Passive_Engagement
Broader pattern: surface compliance without meaningful engagement with correction

Trend across files: clear in CG-W1 and more entrenched in CG-W6; only occasional and minor in EG-W1 through uneven participation or off-task moments

Code reviewed: On_Task_Behaviour
Broader pattern: behavioural order and task compliance, which may or may not involve deep learning
Often appears with: High_Engagement in EG; Low_or_Passive_Engagement in CG
Trend across files: present in both conditions, but in EG it aligns with active processing, whereas in CG it often means orderly but passive behaviour
