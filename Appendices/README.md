# Appendices

## Purpose

This folder contains the dissertation appendices extracted into separate appendix-level folders. Each appendix folder includes a PDF version preserving the original layout and a text version for quick checking, searching, or repository indexing.

## Folder contents

### Direct files

| File | Type | Size |
|---|---:|---:|
| `README.md` | md | 2.0 KB |


### Direct subfolders

| Subfolder | Purpose |
|---|---|
| `Appendix A - Participants` | This appendix contains the anonymized participant lists and baseline information for the experimental and control groups. |
| `Appendix B - Institutional Authorization to Conduct the Study` | This appendix contains the institutional authorization document permitting the study to be conducted at the school site. |
| `Appendix C - The Syllabus Topics` | This appendix contains the syllabus topics used to show alignment between the writing tasks and the second-year scientific-stream English curriculum.. |
| `Appendix D - Rater Participation Letter` | This appendix contains the rater participation letter documenting the scoring role used for the essay assessment process.. |
| `Appendix E - Analytic Rubric` | This appendix contains the essay-scoring rubric used to evaluate overall writing performance across content, organization, vocabulary, grammar, and mechanics.. |
| `Appendix F - Error-Analysis Grid` | This appendix contains the error-analysis grid used to classify and count vocabulary, grammar, and mechanics errors in student essays.. |
| `Appendix G - Validation of the Pre-Test and Post-Test Writing Prompts` | This appendix documents the validation process for the pre-test and post-test writing prompts, including initial prompts, expert feedback, revisions, and final versions.. |
| `Appendix H - Students Essay Samples` | This appendix contains representative anonymized student essay samples from the experimental and control groups. |
| `Appendix I - Observer Invitation Letter` | This appendix contains the invitation letter for the classroom observer/inspector who contributed to the qualitative observation component of the study.. |
| `Appendix J - Classroom Observation Sheet (for main study)` | This appendix contains the classroom observation sheet used for the main study. |
| `Appendix K - Classroom Observation Sheet (for main study)` | This appendix contains an additional classroom observation sheet connected to the study documentation. |


## Recommended use

Use this folder as supporting documentation for the dissertation repository.
